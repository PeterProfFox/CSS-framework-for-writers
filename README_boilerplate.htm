Welcome to MWP

For the first time in the history of mankind MWP brings you bacon-flavoured, armadillo-shaped and kumquat-sized Loxodromes ready to roll straight out of the box.

Using miraculous secrets of ancient-Icelandic technology MWP provides a cure for all known cancers and a handy bottle opener in one simple to use package.

  2. Edit install path/config.cfg and change the user_code
  3. Wait until the last stroke of midnight then run the program. If all goes correctly the devil will appear in a puff of red smoke to claim your soul.
